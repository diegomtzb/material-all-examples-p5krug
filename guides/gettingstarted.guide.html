Getting started with Angular Material

If you need help with creating an Angular app, check out the Angular CLI to get started.

Once you are done setting up with your app or if you already have an existing app, follow these steps bellow to get started.

Step 1: Installing Angular Material


  npm install --save @angular/material @angular/cdk

(Optional) Snapshot builds

A snapshot build from master is also available. Take note that this build may not be considered as stable and may break.


  npm install --save angular/material2-builds angular/cdk-builds
